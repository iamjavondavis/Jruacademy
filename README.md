# Jru Academy

Jru Academy is a responsive, browser-based digital learning app built for Jru.

## Current learning worlds

- Math Galaxy: multiplication missions, mastery checks, locked progression, and a boss battle
- Reading Forest: reading comprehension stories covering main idea, details, vocabulary, sequence, and inference
- Dad Dashboard: XP, mastery scores, progress recommendations, and progress export

## How to use

Open the published GitHub Pages site or open `index.html` directly in a modern browser.

Progress is currently saved in the browser using `localStorage`. This means progress remains on the specific device and browser being used.
